Useful tools for hackers

What is it?

The tools you are going to find here are all open source proejcts. If any of those tools helped you to find bugs, please consider helping them back. You are going to see all projects with their github repositories, give them a star or donate them.

Happy Hacking!

How to use it

Depending on the projects installation, all projects can be download via source code or as seen on their installation guides.

The aim of this page is delivering all the useful tools to hackers out there.

Let us know if you want us to add another tools here, or maybe your tools!
